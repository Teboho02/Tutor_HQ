## ğŸ› What to Do If Something Doesn't Work

### Class Link Won't Open
- Check URL format (should start with https://)
- Make sure you have access rights to the meeting
- Try opening in incognito window

### Can't Upload Test Score Image
- Image must be JPG, PNG, or PDF
- File size must be under 10MB
- Try JPG format if PNG doesn't work

### Retake Button Missing
- Check if tutor enabled retakes (might be disabled)
- Check if you've reached max retakes limit
- Refresh page if button still not showing

### Video Compression Too Slow
- Very large files (1GB+) take longer
- Try "Standard Quality" preset instead of "High"
- Compress on desktop (faster than mobile)
